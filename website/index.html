shopify-scss-autofill
首页 文档 GitHub npm

让 Shopify 页面开发的 SCSS 体验更顺滑

写 SCSS → 安全编译到 assets → Theme Kit 自动上传；同时支持通过 r.resp(pc, mobile, type) 在 PC 端“埋点”，一键生成移动端覆盖样式（per-entry，避免跨页面污染）。

Quick Start 查看全部指令 
npm create shopify-scss-autofill@latest
# 填好 config.yml
npm run dev:theme:auto

核心用途

PC 埋点 → 移动端样式自动生成

在样式里写 r.resp(pc, mobile, type)，工具扫描并生成 _responsive-autofill.<entry>.generated.scss。

安全编译（safe mode）

Sass 先输出到 src/.sass-out/，再同步到 assets/。 默认不会覆盖未知历史 CSS，避免误伤线上资源。

Theme Kit 一键联动

通过 npm run dev:theme:auto 同时跑生成器 + 编译 + theme watch，改完刷新即见。

主要特性

开箱即用

使用 npm create shopify-scss-autofill 直接接入工具链与默认配置。

一键启动

一个命令启动 responsive 生成、CSS 编译、Theme Kit 上传。

可配置

通过 scss-kit.config.json 自定义扫描目录、入口列表、函数名、阈值等。

开始使用

直接看文档：docs。 也可以从 GitHub 获取源码： shopify-scss-autofill。
