渐进式 Shopify SCSS 工作流

面向 Shopify Theme Kit 的 SCSS 工具链

易上手、稳定、可扩展：为日常页面开发提供更顺滑的“写样式 → 编译 → 上传”体验。

Quick Start

PC 埋点 → 移动端样式自动生成

安全编译（safe mode）

Theme Kit 一键联动

你写的 SCSS

.product-card {
  padding: 24px;

  &__title {
    font-size: r.resp(40px, 20px, h1);
    line-height: r.resp(48px, 26px, body);
  }

  .price {
    margin-top: r.resp(24px, 12px, body);
  }

  &:hover {
    &__title {
      letter-spacing: r.resp(0.6px, 0.2px, body);
    }
  }
}

生成的覆盖（示意）

/* Generated by scss-kit */

/* PC（r.resp 转换后，示意） */
.product-card__title {
  font-size: r.clamp_pc(40px, r.min_px(40px, h1, desktop));
  line-height: r.clamp_pc(48px, r.min_px(48px, body, desktop));
}

.product-card .price {
  margin-top: r.clamp_pc(24px, r.min_px(24px, body, desktop));
}

.product-card:hover .product-card__title {
  letter-spacing: r.clamp_pc(0.6px, r.min_px(0.6px, body, desktop));
}

/* Mobile overrides（autofill） */
@media (max-width: 850px) {
  .product-card__title {
    font-size: r.clamp_mb(20px, r.min_px(20px, h1, mobile));
    line-height: r.clamp_mb(26px, r.min_px(26px, body, mobile));
  }

  .product-card .price {
    margin-top: r.clamp_mb(12px, r.min_px(12px, body, mobile));
  }

  .product-card:hover .product-card__title {
    letter-spacing: r.clamp_mb(0.2px, r.min_px(0.2px, body, mobile));
  }
}

开箱即用

通过 npm create 一步接入工具链与默认配置。

一键启动

一个命令同时跑生成器、编译与 Theme Kit 上传。

可配置

用 scss-kit.config.json 自定义扫描目录、入口与阈值。
