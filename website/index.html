面向 Shopify Theme Kit 的 SCSS 工具链

易上手、稳定、可扩展：为日常页面开发提供更顺滑的“写样式 → 编译 → 上传”体验。

Quick Start

开箱即用

通过 npm create 一步接入工具链与默认配置。

一键启动

一个命令同时跑生成器、编译与 Theme Kit 上传。

可配置

用 scss-kit.config.json 自定义扫描目录、入口与阈值。

安全策略

保守的安全策略，极低的误删、误修改风险。

基于 SCSS

流畅、简捷的开发体验。

你写的 SCSS

.product-card {
  padding: 24px;

  &__title {
    font-size: r.resp(40px, 20px, h1);
    line-height: r.resp(48px, 26px, body);
  }

  .price {
    margin-top: r.resp(24px, 12px, body);
  }

  &:hover {
    &__title {
      letter-spacing: r.resp(0.6px, 0.2px, body);
    }
  }
}

生成的覆盖（示意）

/* Final compiled CSS (example) */

/* PC */
.product-card {
  padding: 24px;
}

.product-card__title {
  font-size: clamp(25px, calc(40 * var(--px-to-vw)), 40px);
  line-height: clamp(42px, calc(48 * var(--px-to-vw)), 48px);
}

.product-card .price {
  margin-top: clamp(21px, calc(24 * var(--px-to-vw)), 24px);
}

.product-card:hover .product-card__title {
  letter-spacing: clamp(0.525px, calc(0.6 * var(--px-to-vw)), 0.6px);
}

/* Mobile overrides (autofill) */
@media (max-width: 850px) {
  .product-card__title {
    font-size: clamp(16px, calc(20 * var(--px-to-vw-mb)), 20px);
    line-height: clamp(13px, calc(26 * var(--px-to-vw-mb)), 26px);
  }

  .product-card .price {
    margin-top: clamp(12px, calc(12 * var(--px-to-vw-mb)), 12px);
  }

  .product-card:hover .product-card__title {
    letter-spacing: clamp(0.2px, calc(0.2 * var(--px-to-vw-mb)), 0.2px);
  }
}
