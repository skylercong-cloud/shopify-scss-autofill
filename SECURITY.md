# Security Policy

## Supported versions

This project is currently maintained on the `main` branch.

## Reporting a vulnerability

If you believe you found a security issue:

- Prefer using **GitHub Security Advisories** (private report) if available for this repository.
- Do **not** open a public issue for sensitive vulnerabilities.

When reporting, please include:

- A description of the issue and potential impact
- Steps to reproduce (PoC if possible)
- Affected files/versions

We will acknowledge the report and work on a fix as soon as practical.
